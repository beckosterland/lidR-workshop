# lidR-workshop
This was a workshop presented and demoed to full Graduate program at the Centre of Geographic Sciences during April 2022.
A group of 25 students followed along to learn about using open-sourced tools to explore, analyse and visualize lidar data.
<img width="960" height="540" alt="Intro to lidR" src="https://github.com/user-attachments/assets/c62f61f6-1924-469a-8801-db14356d2347" />

Use the lidr.R file with RStudio to create automated subsetions of Montreal parks and create tree-crown segmentation.
